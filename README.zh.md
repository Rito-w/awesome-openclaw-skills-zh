## 📦 安装

### 方式 1: ClawHub CLI

```bash
npx clawhub@latest install <skill-slug>
```

### 方式 2: 手动安装

将技能文件夹复制到以下位置之一：

| 位置 | 路径 |
|------|------|
| 全局 | `~/.openclaw/skills/` |
| 工作区 | `<project>/skills/` |

### 方式 3: 自动安装

将技能的 GitHub 仓库链接粘贴到 OpenClaw 聊天中，助手会自动处理安装。
